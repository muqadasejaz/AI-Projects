## 📖 Overview

This repository contains my personal collection of beginner-friendly AI practice projects built to strengthen my understanding of modern AI development.

The goal of these projects is not only to learn the theory behind AI systems, but also to gain hands-on experience by building real applications using Large Language Models (LLMs), Prompt Engineering, Retrieval-Augmented Generation (RAG), embeddings, and AI-powered workflows.

Each project focuses on a specific AI concept and serves as a practical exercise to improve my skills in designing, developing, and deploying AI applications. As I progress, this repository will continue to grow with new projects, experiments, and implementations.

These projects are intended for learning, experimentation, portfolio development, and building a strong foundation in AI Engineering.
